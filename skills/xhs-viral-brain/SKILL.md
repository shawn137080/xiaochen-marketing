目标用户：**加拿大华人小老板**（装修、暖通、美容美甲、搬家、园艺割草、补习、清洁、餐饮）

## 一、爆款文案的底层逻辑

小红书用户滑动速度极快，笔记只有 **0.3秒** 在缩略图抢注意力，**3秒** 决定要不要读完。

爆款笔记做对三件事：
1. **钩子** — 标题/首图让人停下来
2. **共鸣** — 正文让人觉得"说的就是我"
3. **行动** — 让人点赞、收藏、评论、私信

## 二、爆款标题公式（必须用其中一种）

### 数字反差型（最高点击）
- `我统计了X天：[意外结论]`
- `[数字]个[目标用户]都在用的[工具/方法]`
- `漏X个电话 = 损失$X，你算过吗`
- `同样开店，为什么他比你多接30%的单`

### 痛点共鸣型（高保存）
- `[工作场景]，那个电话又断了`
- `做着[当前工作]，客人打来的电话...`
- `旺季到了，一个人怎么接得过来`
- `因为没接一个电话，我损失了[X]`

### 悬念好奇型（高点击）
- `加拿大华人老板都在偷偷用的[方法]`
- `我发现同行比我接单快的秘密`
- `来加拿大创业X年，最后悔的一件事`
- `为什么同行装修单比你多？就差这一步`

### 教程干货型（高收藏）
- `[行业]老板必看：X步设置，再也不漏单`
- `华人小生意必装的X个AI工具（附教程）`
- `手把手教你：用AI接电话，5分钟搞定`

### 对比反转型（高分享）
- `有AI接电话 vs 没有：90天数据对比`
- `旺季前 vs 旺季后：就改了这一件事`
- `我以为很贵，用完发现...`

## 三、正文结构（小红书黄金格式）

### 情绪共鸣型结构
```
[开头：1-2句 描述痛苦场景，用"你"代入]
[共鸣放大：再加一句让对方觉得"对对对"]
[转折：我也是这样，直到...]
[解决方案：简短介绍兜兜AI做了什么]
[结果：具体数字或感受]
[行动号召："你们有没有类似经历？评论区聊聊"]
```

### 数据干货型结构
```
[钩子数字开头：大数字单独一行]
[问题背景：这个数字代表什么]
[数据分解：3-5条，每条一行，加 emoji]
[背后原因：为什么会这样]
[解决方案：兜兜AI怎么解决]
[号召："收藏备用，别等漏单了才后悔"]
```

### 工具评测型结构
```
[开门见山：直接说结论，反常识最好]
[使用场景：我在[什么情况]下用的]
[核心功能：① ② ③ 列举（每条简短）]
[真实效果：用前/用后，最好有数字]
[适合谁：[具体行业]的老板]
[号召："想了解的私信我"]
```

## 四、小红书写作规则

### 必须做
- **首句必须是钩子**，不要废话铺垫
- **一句话一行**，多用换行，留白让人喘气
- **数字具体化**：不说"很多"，说"11个"；不说"赚钱"，说"$3,200"
- **用"你"而不是"大家"**，直接对话感
- **结尾必须有互动问题**，引导评论
- **表情符号节制使用**：每段1-2个，不要每句都有
- **地域细节**：加拿大、多伦多、北美、华人社区 — 让目标用户秒认

### 禁止做
- ❌ 开头写"大家好我是..."（废话）
- ❌ 大段文字不换行（没人读完）
- ❌ 过度营销感（"限时优惠！"）
- ❌ 编造功能（只写兜兜AI真实有的：AI接听、短信通知、一键确认、日历预约）
- ❌ 英文混写（华人受众用中文更有亲切感，地名除外）

### 真实功能（只能写这些）
- 24/7 AI接听客户来电
- 通话后30秒内给老板发短信摘要
- 客户可以一键确认预约
- 支持接入Google日历
- 无需自己接，专注手头工作

## 五、爆款配图规则

### 图片类型选择
| 笔记类型 | 配图风格 | 比例 |
|---------|---------|------|
| 情绪共鸣 | 扁平漫画：线条小人漏接电话的挫败感 | 3:4 |
| 数字反差 | 深色数据海报：超大数字居中 | 9:16 |
| 实操教程 | 手绘信息图：步骤卡，米白背景 | 9:16 |
| 行业垂直 | 行业场景漫画：装修/美容/割草 | 3:4 |
| 对比反转 | 双栏对比卡：左暗右亮 | 9:16 |

### 文字位置（重要！）
- **大标题**：左上角 或 画面垂直居中
- **副标题**：标题正下方
- **品牌标识"兜兜AI"**：底部居中，小字，不抢眼
- **禁止**：文字堆在底部两角、右下角角标式标题

### 配色方案（兜兜品牌）
- 主色：珊瑚红 `#FF5C5C` / 暖橙 `#FF8C42`
- 辅色：奶油白 `#FFF9F0` / 深灰 `#2D2D2D`
- 强调：亮黄 `#FFD93D`（数字、重点词）
- 禁用：冷蓝/冷灰系（不符合暖系亲和感）

### 漫画元素（小红书爆款风格）
- 扁平线条小人：简洁轮廓，大圆头，夸张表情（惊讶/崩溃/开心）
- 场景道具：手机响、漏斗、美甲台、装修工具、割草机
- 气泡对话框、箭头、感叹号 — 强调情绪
- 风格参考：微信表情包感 + 小红书手绘卡风格，不要3D/写实

### Gemini Prompt 必须包含
1. 比例（9:16 或 3:4）
2. 扁平漫画/手绘风格说明
3. 文字位置（左上角或居中）
4. 具体文字内容（引号包裹）
5. 配色（珊瑚红 + 奶油白 或 深色背景 + 白字）
6. 负面词：无失真人脸，无复杂背景，无过度渐变，无水印

## 六、各行业专属钩子词库

### 装修/施工
钩子：「工地上没信号」「踩着脚手架」「正在切割的时候」「噪音太大听不见」
痛点：报价客户漏、旺季接不过来、工人在场无暇接电话

### 暖通空调（HVAC）
钩子：「空调季到了」「爬上屋顶的时候」「戴着手套没法接」
痛点：夏冬旺季每天30+ 来电，1个漏接 = 一单装机

### 美容美甲
钩子：「给客人做着指甲」「眼睛上在贴睫毛」「手机震了6次」
痛点：双手都在忙，来电无法接，客人转去别家

### 园艺/割草（Landscaping）
钩子：「推着割草机」「戴着降噪耳机」「春天开始了」
痛点：旺季每天超15个询价电话，接不过来就丢单

### 搬家公司
钩子：「正在搬钢琴的时候」「货车开到一半」「旺季就是6-9月」
痛点：搬家当天根本腾不出手接电话

### 补习班
钩子：「家长报名季打来的」「正在上课的时候」「下午四点涌进来」
痛点：一个家长来电 = $1,500-3,000 价值，漏接就丢了

### 清洁公司
钩子：「客人家里戴着手套」「吸尘器开着」「来电听不清楚」
痛点：差评往往因为 "你们没接我电话"

## 七、互动率提升技巧

### 评论区引爆（必选其一）
- `你们有没有遇到过这种情况？评论区说说 👇`
- `你们[行业]旺季一天大概接几个电话？来测测`
- `觉得有用的收藏，不然刷着刷着就找不到了`
- `有没有用过类似工具的？效果怎么样`
- `你估计自己一个月漏了多少个电话？`

### 保存率提升
- 正文包含可操作的步骤列表
- 数字对比（前后）
- "收藏备用"明示
- 信息密度高（一图/一文 = 多个知识点）

### 私信引流
- `想要[免费试用/价格/设置教程]的私信我`
- `发"接电话"给我，我发你完整攻略`
