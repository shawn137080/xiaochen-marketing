## 第一步：判断笔记类型 → 选图风格

**小红书图片核心任务：在 0.3 秒内让用户停止滑动。**

不同笔记类型，配图目标不同：

| 笔记内容风格 | 配图目标 | 首选图片风格 | 查哪个文件 |
|------------|---------|-----------|----------|
| **情绪共鸣**（我懂你的难） | 让用户觉得"说的就是我" | 真实生活场景 + 人物情绪 | scenes.md |
| **数字反差**（一个震惊你的数字） | 大数字先抓眼球，再引发好奇 | 数字海报 / 深色大字卡 | cards.md |
| **实操攻略**（手把手教你做） | 让用户感觉有干货值得保存 | 手绘信息图 / 步骤卡片 | cards.md |
| **行业垂直**（专门说你这行的） | 让目标用户秒认：这是写给我的 | 行业真实场景 / 氛围图 | scenes.md |

## 第二步：爆款图的 5 个判断标准

选完风格后，在对应文件里找 prompt，用这 5 条判断哪个更爆：

1. **缩略图可读** — 在手机 6cm 宽的格子里，文字/数字清晰可见吗？
2. **有悬念钩子** — 图里有信息缺口让人想点进去吗？（"漏的电话值多少钱？"比"AI接电话"更勾人）
3. **色彩对比强** — 深底白字、或高饱和色块，比低对比灰色系更抓眼
4. **有人/有情感** — 有真实人物/手的图比纯文字/纯产品更有共鸣
5. **真实感 > AI感** — 温暖的手绘风、胶片感照片，比冷光高科技感更适合兜兜品牌

## 第三步：兜兜品牌专属选图指南

兜兜目标用户：加拿大华人小老板（装修、暖通、美容、搬家、园艺等行业）。

### 情绪共鸣类笔记 → scenes.md

搜索关键词：`工作中被打断` / `忙碌场景` / `工匠/师傅` / `小老板`
选标准：画面有人物在专注干活，表情自然真实，暖色调

推荐找：有工作场景的生活照风格（不是棚拍，是街拍/纪实感）

### 数字反差类笔记 → cards.md

搜索关键词：`数字海报` / `大字卡` / `数据可视化`
选标准：深色背景 + 超大字体数字 + 简短说明文字

推荐找：Case 里有 `暗色背景`、`红色大数字`、`简洁排版` 的案例

### 实操攻略类笔记 → cards.md

搜索关键词：`手绘信息图` / `步骤卡` / `操作指南`
选标准：9:16 竖版，米色/暖白背景，红黑草书/手写感字体，2-4个清晰段落

**最强推荐：Case #1（手绘风格信息图卡片）**
```
创作一张手绘风格的信息图卡片，比例为9:16竖版。卡片主题鲜明，背景为带有纸质肌理的米色或米白色，
整体设计体现质朴、亲切的手绘美感。卡片上方以红黑相间、对比鲜明的大号毛笔草书字体突出标题，
吸引视觉焦点。文字内容均采用中文草书，整体布局分为2至4个清晰的小节，每节以简短、精炼的中文
短语表达核心要点。字体保持草书流畅的韵律感，既清晰可读又富有艺术气息。卡片中点缀简单、有趣
的手绘插画或图标，例如人物或象征符号，以增强视觉吸引力，引发读者思考与共鸣。整体布局注意视
觉平衡，预留足够的空白空间，确保画面简洁明了，易于阅读和理解。
```

### 行业垂直类笔记 → scenes.md

搜索关键词：`[行业名称]` 比如 `装修` / `美容` / `园艺`
选标准：真实工作环境感，有行业专属道具（工具、设备、材料）

## 第四步：使用 Prompt

1. 打开对应 .md 文件，搜索关键词找案例
2. 复制完整 prompt（中文或英文均可，Gemini 支持中文）
3. 替换 `[BRACKETS]` 占位符为兜兜具体内容
4. 加上比例参数：`aspect ratio 9:16` 或 `比例9:16竖版`
5. 加负面词：`无水印，无失真人脸，无强烈阴影`
6. 提交给 Gemini API（`gemini-3.1-flash-image-preview`）

## 技术要点

- **9:16 竖版** — 小红书首屏最优比例（卡片、信息图）
- **3:4** — 人像/生活照
- **1:1** — 商品/品牌图
- 结构化 JSON prompt 比自然语言更精准控制排版
- 大多数 prompt 有 `[占位符]`，直接替换即可
